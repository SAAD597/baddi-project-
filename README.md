#  Application Android – Kanban Board

##  Présentation du projet

Ce projet est une application mobile Android de type **Kanban Board**, développée dans le cadre d’un projet universitaire.  
L’objectif principal de cette application est de permettre à un utilisateur de **créer et gérer des projets** à l’aide de tableaux Kanban, facilitant ainsi l’organisation des tâches.

L’application propose une interface simple et intuitive permettant :
- la gestion des utilisateurs,
- la création de projets,
- l’affichage de tableaux (boards),
- la navigation entre différentes pages de l’application.

---

##  Objectifs du projet

- Apprendre le développement Android avec **Java** et **XML**  
- Comprendre l’architecture d’une application mobile  
- Utiliser **Firebase** pour l’authentification et le stockage des données  
- Mettre en pratique la gestion de versions avec **Git et GitHub**  

---

##  Technologies et outils utilisés

- **Android Studio** (environnement de développement)
- **Java** (logique métier)
- **XML** (interfaces graphiques)
- **Firebase Authentication** (gestion des comptes utilisateurs)
- **Firebase Firestore** (base de données NoSQL)
- **Gradle** (gestion des dépendances)
- **Git / GitHub** (gestion du code source)

---

##  Architecture du projet

Le projet est organisé comme suit :

- `java/` : contient les classes Java (Activities, Adapters, Models)
- `res/layout/` : contient les fichiers XML des interfaces graphiques
- `res/menu/` : contient les menus de navigation
- `AndroidManifest.xml` : déclaration des activités et permissions
- `build.gradle` : configuration du projet et des dépendances
